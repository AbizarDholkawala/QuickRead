Getting Started

1

Get your API Key

Visit Google AI Studio to generate a free Gemini API key. It only takes a few seconds.

Open Google AI Studio
2

Paste & Save

Copy your API key, paste it in the field above, and click Save API Key.

3

Test & Verify

Click Test API Key to make sure everything is connected properly.

4

Start Summarizing

Navigate to any webpage, click the QuickRead icon, choose a format, and hit Summarize.

Troubleshooting

Invalid API Key

Your API key appears to be invalid or incorrectly entered.

  • Make sure you copied the entire key
  • Check for extra spaces at the beginning or end
  • Verify the key is from Google AI Studio
  • Try generating a new API key

Quota Exceeded

You've exceeded your API usage quota.

  • Wait a few minutes and try again
  • Check your usage at Google Cloud Console
  • Consider upgrading your API plan

Network Error

Unable to connect to the Gemini API.

  • Check your internet connection
  • Try disabling VPN if you're using one
  • Check if Google API is accessible

Permission Denied

The API key doesn't have the required permissions.

  • Enable the Generative Language API in your Google Cloud project
  • Make sure your API key has access to Gemini models
  • Check API restrictions in Google Cloud Console

About

QuickRead uses the Gemini AI to summarize web content. Your API key is stored securely in your browser's local storage and is never sent anywhere except to Google's Gemini API.
